# Fund Data APIs

This directory contains documentation for AKShare fund-related APIs, organized by fund type.

## Contents

- **[fund_public.md](fund_public.md)** - Public funds (mutual funds, ETFs, etc.)
  - Fund product information, pricing, performance, ratings
  - Historical NAV data
  - Fund manager and company information

- **[fund_private.md](fund_private.md)** - Private funds (hedge funds, private equity, etc.)
  - Private fund product data
  - Performance metrics
  - Private fund market analysis

## Usage

Choose the reference file based on your fund type:
- Use `fund_public.md` for ETFs, mutual funds, and other publicly traded funds
- Use `fund_private.md` for private equity funds, hedge funds, and alternative investments
