# QHKC (Hong Kong/Singapore) Data APIs

This directory contains documentation for AKShare APIs related to Hong Kong and Singapore markets, including Chinese companies listed on those exchanges.

## Contents

- **[index_data.md](index_data.md)** - Hong Kong and Singapore stock indices
  - Index definitions and components
  - Historical index data and performance

- **[fundamental.md](fundamental.md)** - Fundamental analysis data
  - Company financial statements
  - Valuation metrics

- **[broker.md](broker.md)** - Broker and institutional data
  - Broker information
  - Institutional holdings

- **[fund.md](fund.md)** - Fund data on Hong Kong/Singapore exchanges
  - Fund products traded on these exchanges
  - Fund pricing and performance

- **[commodity.md](commodity.md)** - Commodity and derivative data
  - Commodity contracts
  - Futures and options on Hong Kong/Singapore exchanges

- **[tools.md](tools.md)** - Analysis tools and utilities
  - Technical analysis tools
  - Data processing utilities specific to HK/SG markets

## Usage

Select reference files based on your data needs:
- **Index data**: Use `index_data.md` for Hang Seng, FTSE Singapore indices
- **Company fundamentals**: Use `fundamental.md` for financial statements and metrics
- **Market structure**: Use `broker.md` and `fund.md` for holdings and traded products
- **Derivatives**: Use `commodity.md` for commodity and derivative contracts
- **Analysis**: Use `tools.md` for calculation and analysis utilities
